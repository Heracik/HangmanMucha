# Hangman

1. Hráč sa snaží uhádnuť skryté slovo tým, že postupne zadáva jednotlivé písmená.

2. Program po každom pokuse zobrazí, či sa písmeno nachádza v slove, a odhalí jeho pozíciu alebo pripočíta chybu.

3. Pri nesprávnych pokusoch sa postupne kreslí časť postavy na šibenici.

4. Hráč vyhrá, ak uhádne všetky písmená skôr, než urobí maximálny povolený počet chýb.

5. Hra sa skončí prehrou, ak sa nakreslí celá postava a slovo sa nepodarilo uhádnuť.
